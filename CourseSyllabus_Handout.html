១. ការពិពណ៌នាមុខវិជ្ជា (Course Description)

មុខវិជ្ជានេះត្រូវបានរចនាឡើងយ៉ាងពិសេសសម្រាប់និស្សិតដែលទើបឈានជើងចូលក្នុងពិភពនៃការអភិវឌ្ឍប្រព័ន្ធ Back-End និង Application Programming Interface (API) កម្រិតដំបូង។ នៅក្នុងយុគសម័យឌីជីថល កម្មវិធីវិប (Web Application) និងទូរស័ព្ទដៃ (Mobile Application) សុទ្ធតែពឹងផ្អែកលើ API ជាស្ពានចម្លងទិន្នន័យ។ យើងផ្តោតសំខាន់លើស្ថាបត្យកម្មស្តង់ដារ Model-View-Controller (MVC)។

ការយល់ដឹងបែបប្រៀបធៀប៖ API ប្រៀបដូចជា "អ្នករត់តុ" ទទួលបញ្ជាពីអតិថិជន (Frontend Web) យកទៅប្រាប់ចុងភៅ (Database) រួចយកម្ហូប (ទិន្នន័យ) មកជូនវិញ។

ចំណុចពិសេសនៃកម្មវិធីសិក្សា៖ និស្សិតមិនត្រឹមតែចេះកំណត់រចនាសម្ព័ន្ធ Model (មូលដ្ឋានទិន្នន័យ) និង Controller (API Logic) ប៉ុណ្ណោះទេ តែតម្រូវឱ្យសរសេរកូដ Frontend ផ្ទាល់ (HTML & JavaScript) ដើរតួជា View តាមទម្រង់ MVC ដើម្បីអាចហៅទិន្នន័យពី API មកបង្ហាញ និងបញ្ជាផ្ទាល់នៅលើផ្ទាំង User Interface (UI) របស់ Web Application យ៉ាងជោគជ័យ។

សទ្ទានុក្រម និងពាក្យគន្លឹះ (Key Terms)

MVC Architecture

ស្ថាបត្យកម្មសរសេរកូដបែងចែកកម្មវិធីជា ៣ ផ្នែកធំៗ៖ Model (ទិន្នន័យ), View (ផ្ទាំងបង្ហាញ), Controller (អ្នកចាត់ចែង)។

EF Core

បច្ចេកវិទ្យា Object-Relational Mapping (ORM) ដែលជួយឱ្យ Model អាចភ្ជាប់ទៅ Database (DB) តាមវិធីសាស្ត្រ Code-First។

CRUD Operations

ប្រតិបត្តិការគោលទាំង៤ នៃប្រព័ន្ធទិន្នន័យ៖ Create (បញ្ជូល), Read (អាន), Update (កែប្រែ), Delete (លុប)។

CORS Policy

គោលការណ៍សុវត្ថិភាពអនុញ្ញាតឱ្យ Frontend អាចហៅ API ឆ្លង Domain ផ្សេងគ្នាបាន (Cross-Origin Resource Sharing)។

២. លទ្ធផលសិក្សារំពឹងទុក (Course Learning Outcomes)

CLO 1: រៀបចំ Environment ដំឡើងកម្មវិធី VS Code, .NET SDK និងយល់ដឹងពីស្ថាបត្យកម្មគ្រឹះនៃ ASP.NET Core រួមទាំងការសរសេរកូដភាសា C# កម្រិតមូលដ្ឋាន។
CLO 2: អនុវត្តការបង្កើត Controllers តាមទម្រង់ MVC, ការកំណត់ Routing និងការប្រើប្រាស់ HTTP Methods សម្រាប់ទទួលសំណើពី Client ប្រកបដោយភាពត្រឹមត្រូវ។
CLO 3: រចនា និងគ្រប់គ្រងមូលដ្ឋានទិន្នន័យ (Database) តាមរយៈបច្ចេកវិទ្យា Entity Framework Core (EF Core) ដោយប្រើវិធីសាស្ត្រ Code-First Migrations ភ្ជាប់ជាមួយកម្មវិធី SQL Server Management Studio (SSMS)។
CLO 4: បង្កើតមុខងារ Create, Read, Update, Delete (CRUD) យ៉ាងស្ទាត់ជំនាញ ទាំងកម្រិត Backend Web API និងការបង្កើត User Interface (UI) តភ្ជាប់តាមរយៈ JavaScript (JS) Fetch API។
CLO 5: ដោះស្រាយបញ្ហាសុវត្ថិភាព CORS, អភិវឌ្ឍន៍ RESTful API ពេញលេញ និងប្រើប្រាស់ប្រព័ន្ធ GitHub សម្រាប់គ្រប់គ្រងកូដគម្រោងជាក្រុម។

៤. វិធីសាស្រ្តបង្រៀន (Teaching Methods)

ដើម្បីធានាការផ្ទេរចំណេះដឹង និងការអនុវត្តបានជោគជ័យ សាស្ត្រាចារ្យនឹងអនុវត្តវិធីសាស្ត្របង្រៀនដូចខាងក្រោម៖

  • ការពន្យល់ដោយប្រៀបធៀប (Analogy)៖ ប្រៀបធៀបចំណុចបច្ចេកវិទ្យាថ្មីទៅនឹងអ្វីដែលនិស្សិតធ្លាប់ស្គាល់ ដើម្បីបង្កើនល្បឿននៃការយល់ដឹង។
  • ការសរសេរកូដបង្ហាញផ្ទាល់ (Live Coding & Steps)៖ សរសេរកូដបង្ហាញ និងពន្យល់ជំហាននីមួយៗផ្ទាល់នៅលើផ្ទាំង VS Code។
  • ការអនុវត្តផ្ទាល់ស៊ីជម្រៅ (Hands-on Practice)៖ តម្រូវឱ្យនិស្សិតម្នាក់ៗបង្កើតប្រព័ន្ធតភ្ជាប់រវាង Backend API និង Web Page UI ដោយខ្លួនឯង។
  • ការរៀនសហការ និងផ្តោតលើនិស្សិត (Collaborative Learning)៖ ការពិភាក្សាដោះស្រាយបញ្ហា Error, ការសហការតាម GitHub និងការស្រាវជ្រាវគម្រោងផ្ទាល់ខ្លួន។

៥. រង្វាយតម្លៃលទ្ធផលសិក្សា (Assessment & Weights)

សមាសភាគវាយតម្លៃ ទម្ងន់ពិន្ទុ ការពិពណ៌នាលម្អិត

- ករណីចូលយឺត (AL) ចំនួន ៣ ដង ស្មើនឹង អវត្តមាន (A) ចំនួន ១ ដង (១ session)។
