១. ការពិពណ៌នាមុខវិជ្ជា (Course Description)

មុខវិជ្ជានេះត្រូវបានរចនាឡើងយ៉ាងពិសេសសម្រាប់និស្សិតដែលទើបឈានជើងចូលក្នុងពិភពនៃការអភិវឌ្ឍប្រព័ន្ធ Back-End និង Application Programming Interface (API) កម្រិតដំបូង។

នៅក្នុងយុគសម័យឌីជីថល កម្មវិធីវិប (Web Application) និងទូរស័ព្ទដៃ (Mobile Application) សុទ្ធតែពឹងផ្អែកលើ API ជាស្ពានចម្លងទិន្នន័យ។ យើងផ្តោតសំខាន់លើស្ថាបត្យកម្មស្តង់ដារ Model-View-Controller (MVC)។

គំនិតគន្លឹះ៖ API ប្រៀបដូចជា "អ្នករត់តុ" ទទួលបញ្ជាពីអតិថិជន យកទៅប្រាប់ចុងភៅ រួចយកម្ហូបមកជូនវិញ។

១. ការពិពណ៌នាមុខវិជ្ជា (Full-Stack perspective)

ចំណុចពិសេស៖ និស្សិតមិនត្រឹមតែចេះកំណត់រចនាសម្ព័ន្ធ Model (មូលដ្ឋានទិន្នន័យ) និង Controller (API Logic) ប៉ុណ្ណោះទេ តែតម្រូវឱ្យសរសេរកូដ Frontend ផ្ទាល់ (HTML & JavaScript) ដើរតួជា View តាមទម្រង់ MVC។

វិធីសាស្ត្រនេះជួយឱ្យនិស្សិតអាច ហៅទិន្នន័យពី API មកបង្ហាញ និងបញ្ជាផ្ទាល់នៅលើផ្ទាំង User Interface (UI) របស់ Web Application យ៉ាងជោគជ័យ។

សទ្ទានុក្រម និងពាក្យគន្លឹះ (Key Terms) - ភាគទី ១

Model-View-Controller (MVC)

ស្ថាបត្យកម្មសរសេរកូដបែងចែកកម្មវិធីជា ៣ ផ្នែកធំៗ៖ Model (ទិន្នន័យ), View (ផ្ទាំងបង្ហាញ), Controller (អ្នកចាត់ចែង)។

Entity Framework Core (EF Core)

បច្ចេកវិទ្យា Object-Relational Mapping (ORM) ដែលជួយឱ្យ Model អាចតភ្ជាប់ទៅ Database (DB) បានយ៉ាងងាយតាមវិធីសាស្ត្រ Code-First។

សទ្ទានុក្រម និងពាក្យគន្លឹះ (Key Terms) - ភាគទី ២

CRUD Operations

ប្រតិបត្តិការគោលទាំង៤ នៃប្រព័ន្ធទិន្នន័យ៖ Create (បញ្ជូល), Read (អាន), Update (កែប្រែ), Delete (លុប)។

CORS Policy

Cross-Origin Resource Sharing (CORS): គោលការណ៍សុវត្ថិភាពអនុញ្ញាតឱ្យ Frontend អាចហៅ Application Programming Interface (API) ឆ្លង Domain ផ្សេងគ្នាបាន។

២. លទ្ធផលសិក្សារំពឹងទុក (CLO 1 - CLO 3)

CLO 1: ស្ថាបត្យកម្ម API និង C#

រៀបចំ Environment ដំឡើងកម្មវិធី VS Code, .NET SDK និងយល់ដឹងពីស្ថាបត្យកម្មគ្រឹះនៃ ASP.NET Core រួមទាំងការសរសេរកូដភាសា C# កម្រិតមូលដ្ឋាន។

CLO 3: EF Core & SQL Server

រចនា និងគ្រប់គ្រងមូលដ្ឋានទិន្នន័យ (Database) តាមរយៈបច្ចេកវិទ្យា Entity Framework Core (EF Core) ដោយប្រើវិធីសាស្ត្រ Code-First Migrations ភ្ជាប់ជាមួយកម្មវិធី SQL Server Management Studio (SSMS)។

២. លទ្ធផលសិក្សារំពឹងទុក (CLO 4 - CLO 5)

CLO 4: ប្រតិបត្តិការ CRUD Full-Stack

បង្កើតមុខងារ Create, Read, Update, Delete (CRUD) យ៉ាងស្ទាត់ជំនាញ ទាំងកម្រិត Backend Web API និងការបង្កើត User Interface (UI) តភ្ជាប់តាមរយៈ JavaScript (JS) Fetch API។
